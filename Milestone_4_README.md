# AI-Enabled Visa Status Prediction & Processing Time Estimator

**Milestone 4 — Web Application Development & Deployment**

### Objective

The objective of **Milestone 4** is to transform the trained machine learning model into a **fully functional AI-powered web application**.

This stage focuses on:

- building an interactive web dashboard
- integrating the trained ML prediction engine
- creating a scalable API service
- deploying the system to the cloud
- validating the system using real prediction requests
- ensuring production-level reliability, monitoring and fallback mechanisms

The final result is a **production-style AI system** that allows users to estimate visa processing time and receive intelligent insights.

---

# System Architecture

The deployed system follows a **modern AI service architecture** with a separated frontend and backend.

```
User
 │
 ▼
Streamlit Web Dashboard
 │
 ▼
FastAPI Prediction Service
 │
 ▼
Machine Learning Model (Stacking Ensemble)
 │
 ▼
AI Insight Generator (Gemini)
```

This architecture allows the prediction engine to be accessed by **multiple clients**, such as web dashboards, mobile apps, or third-party integrations.

---

# Web Application

The user interface is developed using **Streamlit** to provide an interactive and user-friendly dashboard.

### Dashboard Features

- Visa application input form
- AI-based processing time estimation
- Confidence interval display
- Model confidence score
- Embassy intelligence report generated by AI
- Interactive visualizations for model explainability
- Real-time fallback system (API -> local model)
- Global analytics dashboard with charts and maps

The dashboard communicates with the backend API and automatically switches to a local prediction engine if API fails, ensuring system reliability.

---

# Prediction API Service

A **FastAPI microservice** was implemented to serve predictions through REST API endpoints.

### API Endpoints

| Endpoint  | Method | Description                           |
|-----------|--------|---------------------------------------|
| `/predict`| POST   | Returns estimated visa processing time|
| `/health` | GET    | Checks API service status             |
| `/`       | GET    | API welcome message                   |

API Features

- API key authentication (secure access)
- Input validation using Pydantic
- Dynamic feature engineering inside API
- Model metadata exposure via /health
- Error handling with fallback-safe responses

---

# Processing Time Estimator Engine

The backend prediction engine processes user input and estimates visa processing time.

### Input Parameters

| Parameter        | Description                         |
|------------------|-------------------------------------|
| Fiscal Year      | Application fiscal year             |
| Application Month| Month when application was submitted|
| Region           | Applicant geographic region         |
| Processing Office| Embassy                             |
| Visa Type        | Visa category                       |

### Output

The prediction engine returns:

- Estimated processing time (days)
- Confidence interval (90% uncertainty range)
- Model confidence score
- Benchmark MAE
- Residual interval

The report includes:

This provides users with **actionable insights beyond simple predictions**.
